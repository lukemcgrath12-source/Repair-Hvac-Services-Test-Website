## 🎯 Visual Highlights

### Color Usage
- **Navy (#1E1B4B)**: Primary color for headers, text, dark sections
- **Teal (#0D7A7A)**: CTA buttons, accents, highlights
- **Teal Light (#5bbaba)**: Secondary highlights, hover states
- **White & Off-white**: Content backgrounds, contrast

### Interactive Elements
- Smooth transitions on all interactive elements
- Hover states for buttons (color change, lift effect, shadow)
- Card hover effects (elevation, shadow)
- Focus states for accessibility

### Typography Hierarchy
- H1: Extra-large serif (5-6xl) - hero headline
- H2: Large serif (4-5xl) - section headers
- H3: Medium sans-serif (2-3xl) - card titles
- Body: Regular sans-serif (base-lg) - content
- Caption: Small sans-serif - labels, metadata
