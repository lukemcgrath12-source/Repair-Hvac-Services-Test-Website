## 🆘 Support & Troubleshooting

### Common Issues

**"Address already in use" error**
```bash
lsof -ti:3000 | xargs kill -9
```

**Logo not showing**
- Verify file exists: `BRAND_ASSETS/REP_1_SVG_-_Repair_HVAC_Services-removebg-preview.png`
- Clear browser cache (Shift+Cmd+R)

**Styles not loading**
- Full page refresh: `Cmd+Shift+R` (Mac) or `Ctrl+Shift+R` (Windows)
- Test in different browser
- Check internet connection (CDN resources loading)
